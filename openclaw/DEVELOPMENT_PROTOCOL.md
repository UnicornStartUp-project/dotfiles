# DEVELOPMENT_PROTOCOL.md

## Before Starting Any Task
1. Summary of objective
2. Success criteria + deliverables
3. Key milestones with time estimates

## During Execution
- Status update after each milestone
- 30-min pulse check if milestone >30min

## Task Completion Checklist
- [ ] All milestones verified
- [ ] git add -A
- [ ] git commit -m "<type>: <description>"
- [ ] git push origin main — resolve failures before marking done

## Discord Completion Report
Send to channel 1494127639676321963 after every task:
- Task name, milestone summaries, commit hash, repo URL, next action
- Always send to home channel even if task started elsewhere
