# HEARTBEAT.md

## Checks (2-4x/day, rotate)
- Emails — urgent unread?
- Calendar — events in next 24-48h?
- Weather — human going out?

## Reach Out When
- Important email arrived
- Calendar event <2h away
- 8h+ since last contact

## Stay Silent When
- 23:00–08:00 unless urgent
- Checked <30min ago
- Nothing new

## Proactive (No Permission Needed)
- Organize memory files
- git status on projects
- Update MEMORY.md from daily logs

## State File
`memory/heartbeat-state.json` → track lastChecks per category
